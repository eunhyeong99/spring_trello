# 📘Spring Trello 프로젝트
- 프로젝트 명 : Six Rello 프로젝트
- 소개
    - 한 줄 정리 : 칸반 보드 방식으로 작업을 관리하는 Trello의 백엔드 시스템을 구현하는 팀 프로젝트입니다.
    - 내용 : 이번 팀 프로젝트에서는 프로젝트 관리 툴인 Trello의 백엔드 구현을 목표로 합니다. Trello는 칸반 보드(Kanban Board) 방식을 사용하여 작업을 시각적으로 관리할 수 있는 협업 도구입니다.
